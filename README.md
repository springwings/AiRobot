# AiRobot

#### 介绍 
AiRobot 基于LLMs中文智能交互数字人引擎框架。
 
#### 安装教程
- 根据启动过程报错，自行安装缺失的包，主体包参照requirements.txt
- python3.8.10
- cuda-11.7
- 显存大小 >=40G
- 注意Service/ChatGpt部分，建议自行部署开源LLMs,目前代码实现是阿里通义接口（收费）

#### 使用说明
- 依次运行如下命令：
  - python3 start.py
  - python3 start_avatar.py
